# legendary_money_fabric
Addon de création de fausse money


# Commande disponible : 
Dirtymoney : 
- !dirtymoney : afficher son argent sale
- !paydirty <joueur> <montant> : donner son argent sale à quelqu'un
- !adddirtymoney <joueur> <montant> : ajouter de l'argent sale à quelqu'un
- !removedirtymoney <joueur> <montant> : retirer de l'argent sale à quelqu'un
- !getdirtymoney <joueur> : savoir l'argent sale qu'un joueur possède

Systeme de save de position : 
- !launderingsaveposition : sauvegarde les positions des NPC et des bâtiments
- !launderingdeleteposition : détruit toutes la sauvegarde des positions des NPC et des bâtiments
